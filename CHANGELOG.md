# Changelog

## 0.1.0

- Added a lightweight Hangul/English keyboard correction model.
- Added language-aware correction for TypeScript, JavaScript, Python, JSON, HTML, CSS, Markdown, and plain text.
- Added VS Code commands for correcting selections and whole documents.
- Added automatic previous-token correction for strings, comments, and text regions.
- Added status bar toggle, token ignore rules, always-correct rules, revert-last-correction, debug info, and language overrides.
- Added right-click editor context menu entries.
